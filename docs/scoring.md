## 评分原则

1. 这道题面向初学者，不按高级工程标准打分。
2. 优先看环境配置、API 调用和流式输出是否完成。
3. 只要逻辑基本正确，就不要因为结构不够高级而给过低分。
4. 如果 README 能说明清楚运行方式，可以适当正向考虑。

## 一、环境配置与依赖管理，20 分
- 使用 venv，10 分

## 二、API Key 获取与配置，20 分

- 没有把 Key 硬编码进源码，10 分

- README 说明清楚，5 分
